> 怪我太自私

之前來github建站，在這裡落腳，原本打算做一個瑣碎的生活博客，但是發現github本身是個分享代碼的平台，我覺得不應該浪費這裡的空間做這種對他人沒有意義的博客

於是我在此決定重新命名及定位這個博客存在的意義：

**——Cyzus' Code將致力於分享新穎想法、教程以及應用代碼，不再發表瑣碎無意義的文章**
